# Competitive Programming

Daily CP + DSA practice log. Codeforces (rating-based progression + contests) and LeetCode (interview-style, pattern-based) tracked separately since they train different things.

## Structure

```
competitive-programming/
├── codeforces/
│   ├── 800/        # sheet problems by rating
│   ├── 900/
│   ├── 1000/
│   ├── 1100/
│   ├── 1200/
│   └── contests/    # one subfolder per Div 3/4 contest
├── leetcode/         # interview-style problems, organized by pattern
├── templates/        # reusable C++ snippets (fast I/O, common algorithms)
├── .gitignore
└── README.md
```

## Naming convention

- **Codeforces:** `<contestID><Letter>_<ProblemName>.cpp` — e.g. `4A_Watermelon.cpp`
- **LeetCode:** `<ProblemName>.cpp` — e.g. `TwoSum.cpp`, organized into pattern subfolders inside `leetcode/` as needed (e.g. `leetcode/sliding-window/`)
